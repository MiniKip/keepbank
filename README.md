KeepBank

> Terminal-based banking system in Lua

KeepBank is an open-source, highly responsive, and lightweight terminal-based banking simulation environment written entirely in Lua. Designed for terminal lovers, KeepBank models modern consumer banking operations inside a lightweight, lightning-fast text user interface.

Whether you want to learn Lua, simulate transactional logic, or manage secure CLI-based Ledgers, KeepBank provides a robust foundation.

[!WARNING]
Active Development Warning: KeepBank is currently in an early pre-release phase (underdev). API endpoints, storage schemas, and configuration formats may change significantly between commits.
